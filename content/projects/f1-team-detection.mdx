En este proyecto se implementó un modelo de detección de objetos entrenado con imágenes de monoplazas de F1, utilizando la arquitectura YOLOv8. El modelo fue entrenado con un dataset de imágenes de monoplazas de F1 temporada 2025, y fue capaz de detectar los equipos de F1 con un alto grado de precisión.
